# Nightfall - 宠物生活方式品牌独立站

一个温暖、克制的宠物睡衣品牌网站。

## 🚀 快速开始

### 本地开发

```bash
cd nightfall-store
npm run dev
```

访问 http://localhost:3000

### 部署到 Vercel（推荐）

1. 安装 Vercel CLI：
```bash
npm i -g vercel
```

2. 部署：
```bash
vercel
```

3. 生产环境部署：
```bash
vercel --prod
```

## 📁 项目结构

```
nightfall-store/
├── app/
│   ├── layout.tsx    # 全局布局
│   ├── page.tsx      # 首页
│   └── globals.css   # 全局样式
├── components/        # 可复用组件
├── public/           # 静态资源
└── package.json
```
