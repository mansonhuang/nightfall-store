# Nightfall 多语言功能说明

## 🌐 语言切换功能

### 使用方法

**桌面端：**
1. 页面右上角找到 **🌐 EN/中文** 按钮
2. 点击即可切换语言
3. 页面内容即时更新

**移动端：**
1. 点击右上角汉堡菜单图标
2. 展开菜单底部有语言切换按钮
3. 点击切换语言

### 语言提示横幅

首次访问时会显示顶部提示横幅：
- 中文用户：`🌐 右上角可切换语言`
- 英文用户：`🌐 Language switch available`
- 点击 `×` 可关闭提示
- 导航栏会自动下移避免遮挡

### 视觉反馈

- ✅ 按钮 hover 效果（背景色变化）
- ✅ 当前语言高亮显示
- ✅ 平滑过渡动画
- ✅ Tooltip 提示（鼠标悬停显示）

## 🎨 设计细节

### 按钮样式
```
[🌐 EN] 或 [🌐 中文]
```
- 圆形背景（rounded-full）
- hover 时浅灰色背景
- 图标 + 文字组合
- 字体加粗显示当前语言

### 响应式
- **桌面端**：导航栏右侧显示
- **移动端**：折叠菜单内显示

### 可访问性
- ✅ 键盘可访问
- ✅ 屏幕阅读器友好
- ✅ 触摸目标足够大（44px+）

## 🔧 技术实现

```typescript
const [lang, setLang] = useState<'en' | 'zh'>('zh')
const t = translations[lang]
```

使用 React state 管理语言，即时切换无需刷新页面。

翻译数据结构化存储，易于维护和扩展。
